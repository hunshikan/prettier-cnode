# Prettier CNode

Prettier CNode 是一个更漂亮的响应式（布局）的 CNode 社区，可自适应手机与 PC 屏幕，同时拥有流畅的交互体验。

## 预览链接
[Prettier CNode](http://gwokhou.info/prettier-cnode/dist/index.html)

## 技术栈
- Vue.js
- Vue Router
- ES6
- SCSS
- Flex
- Axios
- Day.js
